Admin Console

Monitor, debug, and manage your LocalMCP instance through the web-based admin console.

Accessing the Admin Console

The admin console is available at http://localhost:3001 when enabled.

Enable Admin Console

Add this to your localmcp.config.json:

{ "admin": { "enabled": true, "port": 3001, "auth": { "username": "admin", "password": "your-secure-password" } } }

API Endpoints

The admin console exposes several API endpoints for programmatic access:

  • GET /api/health - System health check
  • GET /api/metrics - Performance metrics
  • GET /api/services - Service status
  • GET /api/cache - Cache statistics
  • GET /api/logs - System logs
  • GET /api/tools - Tool usage statistics
  • GET /api/analytics - Learning analytics
  • GET /api/lessons - Lessons learned data
  • GET /api/learning - Learning metrics

Security

The admin console includes several security features:

  • Authentication - Username/password protection
  • HTTPS Support - Secure communication
  • Access Logging - Track who accesses the console
  • Rate Limiting - Prevent abuse
