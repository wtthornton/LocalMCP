What is LocalMCP?

LocalMCP is a local Model Context Protocol (MCP) server that provides AI assistance for coding tasks. It's designed for developers who prefer AI assistance and simple setups.

üõ†Ô∏è How to Use LocalMCP

1. Check System Health

curl http://localhost:3000/health

2. Access MCP Tools via JSON-RPC

# Initialize MCP connection curl -X POST -H "Content-Type: application/json" -d '{"jsonrpc": "2.0", "id": 1, "method": "initialize", "params": {"protocolVersion": "2024-11-05", "capabilities": {}, "clientInfo": {"name": "my-client", "version": "1.0.0"}}}' http://localhost:3000/mcp

3. List Available Tools

# Get list of MCP tools curl -X POST -H "Content-Type: application/json" -d '{"jsonrpc": "2.0", "id": 2, "method": "tools/list", "params": {}}' http://localhost:3000/mcp

4. Use MCP Tools

# Analyze code curl -X POST -H "Content-Type: application/json" -d '{"jsonrpc": "2.0", "id": 3, "method": "tools/call", "params": {"name": "localmcp.analyze", "arguments": {"target": "my-code.js", "analysisType": "code"}}}' http://localhost:3000/mcp
