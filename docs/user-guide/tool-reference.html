Tool Reference

Complete reference for all LocalMCP tools and their capabilities.

localmcp.create

Creates new files, components, or code based on your specifications.

Parameters

  • description - What you want to create
  • framework - Target framework (React, Vue, Angular, etc.)
  • language - Programming language (TypeScript, JavaScript, etc.)
  • styling - CSS framework (Tailwind, Styled Components, etc.)

Example

{ "tool": "localmcp.create", "parameters": { "description": "A responsive navigation bar with mobile menu", "framework": "React", "language": "TypeScript", "styling": "Tailwind CSS" } }

localmcp.analyze

Analyzes existing code and provides insights about structure, patterns, and potential issues.

Parameters

  • targetPath - File or directory to analyze
  • analysisType - Type of analysis (structure, performance, security, etc.)
  • depth - Analysis depth (shallow, medium, deep)

Example

{ "tool": "localmcp.analyze", "parameters": { "targetPath": "src/components", "analysisType": "structure", "depth": "deep" } }

localmcp.fix

Identifies and fixes errors, bugs, and issues in your code.

Parameters

  • targetPath - File containing the issue
  • errorDescription - Description of the error or issue
  • fixType - Type of fix (syntax, logic, performance, etc.)

Example

{ "tool": "localmcp.fix", "parameters": { "targetPath": "src/utils/helpers.ts", "errorDescription": "TypeScript error: Property 'name' does not exist on type 'User'", "fixType": "syntax" } }

localmcp.learn

Captures patterns, solutions, and best practices for future reference.

Parameters

  • pattern - The pattern or solution to remember
  • context - Context where this pattern applies
  • category - Category of the pattern (authentication, UI, API, etc.)

Example

{ "tool": "localmcp.learn", "parameters": { "pattern": "JWT token validation middleware", "context": "Express.js API authentication", "category": "authentication" } }
