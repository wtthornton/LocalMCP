Troubleshooting

Common issues and their solutions when using LocalMCP.

Installation Issues

Docker Container Won't Start

Symptoms: Container exits immediately or fails to start

Solutions:

  • Check Docker is running: docker info
  • Verify port 3000 is available: netstat -an | grep 3000
  • Check container logs: docker logs <container-id>

Context7 Connection Issues

Symptoms: "Failed to connect to Context7" errors

Solutions:

  • Verify your Context7 API key is correct
  • Check internet connectivity
  • Ensure Context7 service is available

Performance Issues

Slow Response Times

Symptoms: Requests take longer than expected

Solutions:

  • Check system resources: docker stats
  • Monitor vector database performance
  • Clear cache if it's too large
  • Restart the container

High Memory Usage

Symptoms: Container uses excessive memory

Solutions:

  • Reduce cache size in configuration
  • Limit vector database memory usage
  • Restart container periodically

Tool-Specific Issues

Create Tool Not Working

Symptoms: Files not being created or created incorrectly

Solutions:

  • Check file permissions in mounted volume
  • Verify target directory exists
  • Check for conflicting files

Analyze Tool Returns Empty Results

Symptoms: Analysis returns no insights

Solutions:

  • Verify target path exists and is readable
  • Check if files contain valid code
  • Ensure proper file extensions

Getting Help

If you're still experiencing issues:

  • Check the Admin Console for detailed logs
  • Enable debug mode in your configuration
  • Collect system information and error logs
  • Create an issue on the project repository
