Troubleshooting

Common issues and solutions for PromptMCP.

Service Not Starting

# Check if port 3000 is available
netstat -an | grep 3000

# Kill any process using port 3000
lsof -ti:3000 | xargs kill -9

# Restart the service
npm start

Context7 Configuration Issues

If Context7 is not responding, check your API key and network connectivity. Run npm run test:context7 to verify integration.

Pipeline Errors

Check the admin console for detailed pipeline execution logs and error messages.
